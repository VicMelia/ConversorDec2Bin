# ConversorDec2bin
 
 ## Instrucciones del código del archivo ConversorDec2bin.py
	
	1. Insertar un número decimal positivo. 
	2. Insertar el número de bits determinado para pasarlo.
	3. Convierte el número decimal positivo elegido a binario, usando el nombre de bits que hemos insertado en el paso anterior.
	4. Muestra por pantalla el resultado obtenido, consiguiendo así convertir el número decimal positivo a binario.
